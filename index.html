Materi yang Bisa Dipelajari

Kamu bebas pilih mau fokus ke bidang apa. Semua materi diajarkan dari nol sampai bisa bikin project sendiri.

HTML & CSS

Fondasi website. Belajar struktur halaman, layout, warna, font, dan membuat website responsif untuk tugas sekolah atau portofolio.

Laravel (PHP)

Framework web paling populer. Cocok untuk belajar bikin sistem informasi, toko online, atau aplikasi manajemen data yang powerful.

Flutter (Mobile Apps)

Bikin aplikasi Android & iOS sekaligus dengan satu bahasa (Dart). Tampilan modern, smooth, dan bisa diupload ke PlayStore.

Java

Bahasa klasik & kuat. Dipakai untuk dasar pemrograman berorientasi objek (OOP), aplikasi desktop, dan dasar pengembangan Android.

Kotlin

Bahasa resmi Google untuk Android. Lebih singkat & aman dari Java. Cocok untuk yang ingin fokus bikin aplikasi Android kekinian.

C++

Asah logika & algoritma. Sangat direkomendasikan untuk persiapan Olimpiade Informatika (OSN/KSN) dan memahami cara kerja komputer.

💡 Tips: Disarankan mulai dari HTML/CSS atau C++ dulu untuk membangun dasar logika, baru lanjut ke framework/mobile app.
